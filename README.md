# TXTAMP

> It really whips the text llama's ass.

`txtamp` is a terminal client for Navidrome, inspired by the simplicity and speed of old-school Winamp.

## Features

- Browse artists, albums, and tracks
- Stream music from Navidrome
- Keyboard-first interface
- Fast startup
- Lightweight

## Tech Stack

- Go
- Bubble Tea
- Lip Gloss
- Bubbles
- mpv

## Themes

Press `t` in the app to switch themes.

You can also set `TXTAMP_THEME` before launching:

```sh
TXTAMP_THEME=violet-terminal go run .
```

Built-ins: `default`, `violet-terminal`, `neon-grid`, `phosphor-amber`, `deep-space`, `paperwhite`, `monolith`.

Aliases: `retro`, `sci-fi`, `futuristic`, `amber`, `dark`, `light`, `mono`.
